Javascript is a high-end intrepreted programming language that allows developers to add dynamic interactivity to web pages. 

### Where is it used?

- **Front-end**: Adding interactivity to websites (e.g., buttons, animations, pop-ups).
- **Back-end**: Using environments like Node.js for server-side development.
- **Mobile apps**: Frameworks like React Native.
- **Game development**: Using libraries like Phaser.
- **Desktop apps**: Platforms like Electron.

- **What you can make**:  
    - Websites and web apps.
    - Mobile apps.
    - Games (2D/3D).
    - Desktop applications.
    - Server-side applications (APIs, databases).

- **Origins**:
    - Created in 1995 by Brendan Eich at Netscape.
    - Initially called "Mocha," then renamed "LiveScript," and finally "JavaScript."
    - Standardized as ECMAScript (ES) in 1997.

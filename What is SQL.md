**SQL** (Structured Query Language) is a programming language used to manage and manipulate relational databases. It allows you to perform tasks like querying, updating, inserting, and deleting data.

### Key Points:
- **Relational**: Data is stored in **tables** (rows and columns). 
- **Structured**: Requires a fixed schema (table structure).
- **Used for**: Managing structured data in databases (e.g., MySQL, PostgreSQL).

### Common SQL Operations:
- **SELECT**: Retrieve data.
- **INSERT**: Add new data.
- **UPDATE**: Modify existing data.
- **DELETE**: Remove data.

SQL is widely used in traditional databases that require complex queries and relationships between data.
